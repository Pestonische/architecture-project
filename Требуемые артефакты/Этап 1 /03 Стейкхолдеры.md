## 3. Стейкхолдеры

| Стейкхолдер | Интересы |
|------------|----------|
| Руководство компании | Рост вовлечённости и узнаваемости бренда |
| Маркетинг | Аналитика, персонализация, эксперименты |
| Пользователи | Мотивация, удобство, безопасность данных |
| Партнёры | Интеграции и совместные активности |
| Команды разработки | Гибкая архитектура, технологическая свобода |
| DevOps / SRE | Надёжность и масштабируемость |
| Security / Legal | Соответствие требованиям защиты данных |
